单例模式的特点
  • 单例类只能有一个实例。
  • 单例类必须自己创建自己的唯一实例。
  • 单例类必须给所有其他对象提供这一实例。
  • 单例模式保证了全局对象的唯一性，比如系统启动读取配置文件就需要单例保证配置的一致性。

java.lang.Runtime就是一个单例模式
